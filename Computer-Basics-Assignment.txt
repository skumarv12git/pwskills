Computer Basics Assignment

1. What is a Computer
Ans: Computer is a electronic device which takes instructions from the user and processes it to give desired output.

2. What is RAM?
Ans: RAM full form is Random Access Memory. RAM is also know as Main / Primary memory. Is is made with semiconductor technology. RAM acts as intermediary device between HDD (Hard Disc Drive) (secondary memory) and CPU (processor) in communicating the instructions from and to the user. RAM is very fast in transmitting the instructions from HDD to CPU. When power supply goes off, the data within the RAM is completely cleared.

3. Where is data stored in a computer?
Ans: Data is stored in HDD (Hard Disc Drive).

4. What is the input device used to type text and numbers on a document in a computer system?
Ans: Keyboard.

5. What are output devices?
Ans: Output devices are those which provide processed data back to the user. Example - printer, monitor

6. What is the input device that allows a user to move the cursor or pointer on the screen?
Ans: Mouse

7. Which language is directly understood by the computer without a translation program?
Ans: Machine language. 0s and 1s

8. What are input devices?
Ans: Input devices are those which are used to give instructions to an electronic device to process the same. Example - Keyboard, Mouse, USB
